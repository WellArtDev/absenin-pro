### Key Decisions

1. **Visual vibe: Tech-startup modern** — user explicitly chose modern tech-startup over korporat. Clean, whitespace generous, no heavy gradients.

2. **Primary color: Emerald green (#059669)** — user chose green, PM picked emerald shade. Fresh, professional, trust-signaling. Meets WCAG AA contrast on white.

3. **Typography: Inter** — clean modern sans-serif, excellent legibility in Indonesian. Single font family for both heading and body.

4. **Component library: None** — vanilla CSS custom properties (web) + Flutter ThemeData (mobile). No third-party UI library. Matches constraint: lightweight, no build step needed.

5. **Navigation**: Mobile: bottom tab bar (5 tabs). Web: sidebar persistent + breadcrumb.

6. **Map: Leaflet.js + OpenStreetMap** — free, no API key, consistent with PRD constraint.

7. **Search: Debounce 300ms** — no submit button, auto-search. Consistent with PRD NFR §5.2.

8. **Working mode: Fast path** — user chose fast path. Spines drafted with all decisions inline. Cross-checked against PRD user journeys (UJ-1 through UJ-5).

9. **IA**: 6 mobile surfaces, 9 web dashboard surfaces. Every PRD feature has a surface.
